현재까지의 작업 내용을 CLAUDE.md에 반영하고 커밋/푸시하는 명령어입니다.

다음 순서로 진행하세요:

1. **CLAUDE.md 업데이트**
   - 현재 대화에서 진행된 작업 내용을 반영
   - E2E 테스트 통과 현황 업데이트
   - 완료된 파일 목록 업데이트
   - 다음 작업 목록에서 완료된 항목 제거
   - 새로운 설계 결정 사항이 있으면 추가
   - 현재 진행 상태 업데이트

2. **변경된 소스 파일 확인**
   - `git status`와 `git diff`로 변경사항 확인
   - 변경된 소스 코드 파일이 있으면 함께 커밋 대상에 포함

3. **커밋 & 푸시**
   - 변경된 파일들을 staging
   - 커밋 메시지는 기존 컨벤션을 따름 (한글, `feat:` / `docs:` / `fix:` 등 prefix 사용)
   - `git push`로 원격 저장소에 반영
